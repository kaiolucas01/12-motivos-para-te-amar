❤️ 1 Ano, 12 Motivos, Um Amor
Este é um projeto web simples e emocionante desenvolvido para celebrar 1 ano de relacionamento. O site funciona como um mural digital, exibindo 12 momentos especiais e mensagens de carinho através de um layout responsivo.

🚀 Sobre o Projeto
O objetivo deste projeto foi criar uma surpresa personalizada e interativa. Utilizei o Bootstrap 5 para garantir que o mural de fotos e mensagens (bilhetes) ficasse organizado e bonito tanto no computador quanto no celular.

🛠️ Tecnologias Utilizadas
HTML5: Estrutura do conteúdo.

CSS3: Estilização personalizada (arquivo style.css).

Bootstrap 5: Framework para layout responsivo e componentes de cards.

Google Fonts / Fontes padrão: Para uma leitura agradável.

📸 Funcionalidades
Mural de Memórias: 12 cartões estilizados contendo fotos e legendas.

Design Responsivo: O layout se ajusta automaticamente (3 colunas em telas grandes, 1 coluna em telas de celular).

Rodapé Personalizado: Uma assinatura dedicada com carinho.

📂 Estrutura de Pastas
Para que o projeto funcione corretamente, certifique-se de que a estrutura de arquivos esteja assim:

Plaintext

/
├── index.html       # Arquivo principal
├── style.css        # Estilização do cabeçalho, cores e fontes
└── imagens/         # Pasta contendo as fotos (foto1.jpg até foto12.jpg)
